# curly-sniffle 
## tweet-sentiment analysis

## clone this repo
```
git clone https://github.com/cmucheru/curly-sniffle.git
```

## navigate to installation folder
```
cd curly-sniffle
```

## install the libraries

```
 pip install pandas numpy wandb
```
## run 
```
 python model.py
```
